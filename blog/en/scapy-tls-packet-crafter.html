Scapy-TLS-Packet-Crafter: Custom TLS 1.2 Handshake Generation

By Shay Mordechai | June 17, 2025

Developed during reserve duty, this 4,030-line Python project simulates a full TLS 1.2 handshake from scratch using Scapy and Cryptography, bypassing standard OS SSL libraries[cite: 7].

Core Implementation

1. Client Hello & SNI: Secure generation of the 32-byte client random and building Server Name Indication extensions directly at the byte level[cite: 7].

2. Master Secret PRF: Executing the TLS 1.2 Pseudo-Random Function (HMAC-SHA256) to derive the 48-byte master secret from the pre-master secret and random seeds[cite: 7].

3. Application Data Encryption: Encrypting data using AES-128-CBC. Implemented HMAC-SHA256 for Data Integrity over the sequence number and injected PKCS#7 Padding to meet AES block requirements[cite: 7].

Generated runtime `SSLKEYLOGFILE` traces to debug decryption failures within the Wireshark Dissector, achieving 100% protocol compliance[cite: 7].
